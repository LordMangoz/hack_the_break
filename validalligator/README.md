
## Features

This is a extension for beginner and intermedeate coders that can help them with debugging, refracting, and suggestions.
It's main features is a highliter that highlights HTML format errors and gives detailed explanations. It's sub features are AI that can read highlighted code, provide suggestions, and save to a file.

To start the extenstion simply press the play button, and to enable the AI features press the enable AI button.

Commands to remember:
Debug
Hightlight Warnings
Change note.md directory
Take note
Update number of hashtags headers for timestamps for note.md
Suggestions
Refactor
SET API KEY
Change Extension Name
Change File Name


## Requirements
- npm install @google/genai
- npm install marked
- npm install dotenv
